框架介绍

Spring Web MVC是一种基于Java的实现了Web MVC设计模式的请求驱动类型的轻量级Web框架，即使用了MVC框架模式的思想，将Web层进行职责解耦，基于请求驱动指的就是使用请求-响应模型，框架的目的就是帮助我们快速开发

框架优点

1.设计层次结构清晰
2.降低了各个模块之间的耦合性
3.提高代码的复用性
